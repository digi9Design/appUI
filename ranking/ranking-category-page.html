docs/ui-structure/ranking/ranking-category-page.md

结构说明

顶部高度与屏宽比 `videoHeight = width * 1334/750` 的区域内：`RankingBackgroundWidget`（**静态**底层 `rankingHeaderBgImage(type)` + **叠层** `rankingBgFrontImage(type)`：`ranking_bgfront_giftrecv` / `ranking_bgfront_giftsent` / `ranking_bgfront_room` / `ranking_bgfront_family`，贴背景底居中、宽与背景一致、高随图比例，无视频）。

叠加大横条 `ranking_header_trans_bg`；下方白底占位 `Column` + `CustomScrollView`。

上半 `Stack`：日期 segment（`RankingPageSubtypeSegmentButton`：**外层半透明白胶囊 + 选中纯白滑块 + 选中字色按分类** giftrecv `#FF5576` / giftsent `#FF7A55` / room `#555BFF` / family `#5596FF`，未选白字，无切图）、前三名 `RankingMedalAvatar`（姓名无黑色半透明底）；前景层 `RankingFrontAnimWidget` 当前为空占位。

Header 与列表之间**无** `ranking_div_bg` 分界线图，直接白底列表衔接。

列表区**白底** + `useLightCellStyle` 的 Cell；底部 `myCell` 白底（无 `ranking_family_bottom_bg` 图）。
