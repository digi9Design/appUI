结构说明

顶部 AppBar：`Function switch`

开关列表：

`Room foreground service`：Android 端展示

`Message reminderin room`

`Room gift effects`

`Automatic opening mic`

`Game banner notification`

`Lucky Gift Separate Calculation`

`No-Stranger-Message Mode`

`Lucky Gift Effect`：弹出 High / Normal / Low action sheet

`Filter`：弹出 Girls only / Boys only action sheet

底部 Cancel / Confirm
