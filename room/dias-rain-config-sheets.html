结构说明

根节点：`SafeArea` 包裹固定高度 `dp(614)` 的 `Container`，宽度为当前屏幕宽度。

背景：`BoxDecoration.image` 使用 `R.image.dias_rain_config_bg_img()`，`fit: BoxFit.cover`。

内容主轴为 `Column`：

顶部预留 `dp(123)`，标题区域当前注释掉 `_titleWidget()`，再预留 `dp(45)`。

配置区域宽 `dp(345)`、高 `dp(349)`，左侧内边距 `dp(7)`，内部纵向排列三组配置项。

底部确认按钮与配置区域间距 `dp(24)`，按钮尺寸 `dp(228) x dp(46)`，圆角 `dp(36)`。

金额区 `_diasAmountWidget`：

标题使用 `S.current.k_dias_amount`。

网格宽 `dp(291)`、高 `dp(162)`，2 列，横向间距 `dp(24)`，纵向间距 `dp(16)`。

单元格尺寸 `dp(133) x dp(42)`，含 `icon_greendias_60` 和格式化后的金额文本。

可领取人数区 `_numWidget`：

标题使用 `S.current.k_claimable_slots`。

横向单行网格宽 `dp(345)`、高 `dp(36)`，间距 `dp(10)`。

单元格尺寸 `dp(47) x dp(36)`，显示人数数字。

开奖时间区 `_timeWidget`：

标题使用 `S.current.k_dropping_time`。

横向单行网格宽 `dp(345)`、高 `dp(36)`，间距 `dp(10)`。

单元格高度 `dp(36)`，水平 padding `dp(18)`；`time == 0` 显示 `S.current.k_at_once`，否则显示 `'$time mins'`。
